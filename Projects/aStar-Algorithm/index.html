C++ A* Algorithm

Pathfinding Algorithm

Team Members: Jared Grounds, Josh Jones, Kenny Park, Susan Su

Description: As pathfinding AI becomes more prominent in real-world navigation, mathematics, and video games, it is necessary for one to visualize the logic behind these algorithms. Our team will be studying the popular A* Algorithm, published by Peter Hart, Bertram Raphael and Nils Nilsson, then creating an application that allows a user to build their own scene and run the algorithm in order to find the shortest path from one point to another.
